# 결과 출력 규칙 상세

## 1. 기본 원칙

- **카드형·대시보드형·인터랙티브 UI 아티팩트 사용 금지**
  - 드롭다운, 탭, 필터 버튼 등 포함
- 보고서형 마크다운 본문 중심 → ss-report 연계 시 본문 재활용 가능
- 결과 3건 이상 → 반드시 마크다운 표 (HTML 테이블 아티팩트 금지)

### ★ 출력 순서 원칙

> **어떤 요청이든 스크립트 결과는 표를 먼저 출력한 뒤, 상세 서술을 이어간다.**
> 상세 서술이 필요한 경우에도 표 생략 없이 표 → 서술 순서를 지킨다.

```
1. [검색 해석]
2. 결과 요약 표 (기업명·분야·지역·현재 상태·투자년도·누적투자금액 등 핵심 컬럼)
3. 상세 서술 (의도 해석에 따라 깊이 조절)
4. 차트 (해당 시)
5. 후속 제안
```

**표에 포함할 기본 컬럼 (search_portfolio 결과 기준):**
- 기업명 / 분야 / 지역 / 현재 상태 / 투자년도 / 누적 투자금액(M$)
- 질문 맥락에 따라 라운드·지분율·투자조건 등 추가

**표 생략이 허용되는 경우:**
- 결과 2건 이하
- 사용자가 "설명만", "서술로", "표 없이" 명시한 경우

## 2. 텍스트 서술 규칙

### 개조식 표현

- 서술형 문장 대신 핵심만 추린 개조식(~임, ~함, ~예정)
- 항목 2개 이상 → 글머리 기호(`-`) 목록
- 한 항목 2줄 이내, 내용 많으면 하위 항목으로 분리

### 헤더 구조화

- 복수 기업 다룰 때 → 기업별 `###` 헤더
- 한 기업 내 여러 주제(뉴스, 재무, 현황 등) → `####` 헤더
- 집계 결과 → 헤더 없이 표 바로 출력

### 적용 예시 (뉴스 서술)

```markdown
### Acme Corp
#### 최근 뉴스 (출처: CB Insights)
- **Series C $50M 클로징** (2026-04-15): Sequoia 주도, 기업가치 $400M 평가
  - 당사 지분 희석 예상 → 후속 투자 여부 검토 필요
- **CTO 교체** (2026-03-28): John Kim → Sarah Park 선임, AI 제품 고도화 배경
  - 기술 전략 변화 모니터링 필요
```

## 3. 뉴스 날짜 필수 명시

- 형식: `(YYYY-MM-DD)` — 기사 헤드라인 뒤 괄호 안
- 날짜 확인 불가 시: `(날짜 미확인)` 표기 — **생략 금지**

```markdown
# 올바른 예
- **Series C 클로징** (2026-04-15): ...내용...

# 잘못된 예
- Series C를 클로징했다.   ← 날짜 없음
```

## 4. 수치 시각화

차트는 **별도 아티팩트 (단독 차트)** 로 출력. 카드형 레이아웃 안에 넣지 않는다.
차트는 항상 표 다음에 위치한다.

### 차트 반드시 출력 (판단 없이 실행)

- `get_statistics` 실행 결과 (집계값 포함 시)
- `get_portfolio_summary` 실행 결과
- 시계열 데이터 포함 시 (연도별 추이 등)

### 차트 생략

- `search_portfolio` 단독 결과
- `get_company_history` 결과 (단일 기업)
- 결과 2건 이하
- 해당 요청에서 "표만", "간단히", "결과만" 명시 시
  (이전 요청의 간소화 지시는 이후 요청에 이어지지 않음)

### 차트 유형

| 데이터 성격 | 차트 유형 |
|---|---|
| 크기·순위 비교 | 가로 막대 |
| 시계열 | 꺾은선 |
| 구성 비율 | 파이·도넛 |
| 복수 지표 | 복합 차트 |

**렌더링 오류 시:** 1회 재시도 → 실패 시 표로 대체 + 오류 고지

## 5. Claude 의견·분석 포함 시

DB 조회 결과 + 외부 데이터 + Claude 판단이 혼합된 결과물에는 반드시:

- 각 항목마다 선정·판단 이유 1~2문장으로 명시
- 근거 출처 구분 표기:
  - `(근거: 포트폴리오 DB — 누적 투자 $5M, 3라운드)`
  - `(근거: CB Insights — 최근 Series B $30M 클로징)`
  - `(근거: Claude 판단 — 섹터 성장성 및 팀 구성 고려)`

## 6. 후속 제안 상세

### CB Insights 연동 제안 실행 시

- 5개사 초과 → 전체 조회 여부 먼저 확인
- 정보 없으면 "CB Insights에 해당 기업 정보가 없습니다" 명시
- DB 데이터 혼합 시 출처 구분:
  - `(출처: 포트폴리오 DB)` / `(출처: CB Insights)`

### 그룹별 요약

기업 목록 조회 결과에서 그룹별 요약이 실질적으로 도움이 된다고 판단될 때만 추가한다.
